### 像数 1, 2, 3 一样容易

<CodeGroup>
  <CodeGroupItem title="NPM">

```bash
# 在你的项目中安装
npm init @winner-fed/project@2 my-project

# 进入工程
cd my-project

# 安装依赖
npm run bootstrap

# 开发调试
npm run serve

# 构建打包
npm run build
```
